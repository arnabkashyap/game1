# Escape Road 3D | Infinite Pursuit

A high-performance 3D infinite runner car game built with Three.js.

## Features
- Dynamic difficulty scaling
- Wanted level system (0-5 stars)
- Police pursuit AI (ahead and behind)
- Global multiplayer sync with Socket.IO
- Responsive touch controls for mobile

## Development
- **Local**: `npm start`
- **Build APK**: Uses GitHub Actions with Capacitor
